React is a powerful and influential Javascript library. While working through the Free Code Camp Full Stack Javascript curriculum, I decided to build these 25 projects using React. Nearly half are actually projects from the FCC curriculum, in fact I re-built all 8 of the major Front End Projects (see that project's Github here for an interesting comparison of my 1st and 2nd attempts).

While building these projects, I enountered many resources in the process of learning React and the React ecosystem. Here I've compiled a list of the most useful resources I came across in order to to help anyone else who might be aspiring to learn more about React. There are many excellent resources available online, so I encourage you to find what suits your learning style. However, I can recommend the following (organized roughly in order of beginning to advanced level).

Thinking in React

A great place to start. All of the Facebook documentation is excellent and a great resource to keep handy.

Thinking in React Screencast Tutorial

A very nice and short Youtube tutorial by Hendrik Swanepoel of Tagtree demonstrating many of the 'Thinking in React' principles.

React for Everyone

LevelUpTuts Video Tutorial Series introducing you to React. This is a great place to start with the basics.

Basic React Tutorial Series

Awesome free video series by Joe Maddalone on Egghead.io walking you through a number of basic React demonstrations. You begin working with webpack and babel in the series, which I recommend, but you can also quickly start writing React in Codepen.

9 things every React.js beginner should know

Some React best practices by full stack developer Cam Jackson.

Presentational and Container Components

Dan Abramov on defining components in React.

ReactJS Program

Tyler McGinnis has created these excellent React tutorials, the first of which is free. I cannot recommend these resources enough.
